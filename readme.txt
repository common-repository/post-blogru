== Description ==
Plugin automatically publishes posts from the wordpress blog on a service Blog.ru. The plugin sends the letter to service Blog.ru.<br>Download russian localization <a href="http://bogutsky.ru">this</a>.<br>
Плагин автоматически публикует записи с вашего блога на wordpress на сервисе Blog.ru. Плагин отсылает письмо на Blog.ru.<br>Скачать русскую локализацию можно <a href="http://bogutsky.ru">здесь</a>.

== Installation ==

1. Upload plugin files to the `/wp-content/plugins/` directory.<br>
2. Activate the plugin through the 'Plugins' menu in WordPress.<br>
3. Download russian localization <a href="http://bogutsky.ru">this</a>.<br>
<br>
1. Загрузите файлы плагина в директорию `/wp-content/plugins/`.<br>
2. Активируйте плагин в панели администратора.<br>
3. Скачать и установить русскую локализацию <a href="http://bogutsky.ru">здесь</a>.


== Frequently Asked Questions ==
= How do I set a publish my posts on Blog.ru? = 
Pass in the admin menu and set user login and code word by account Blog.ru.

= Как настроить публикацию на Blog.ru? = 
Введите в панели администратора данные аккаунта Blog.ru.
